# 三传
三传，用来描述事情的发展过程。初传也叫发用，中传为转移，末传为结尾。
## 问题：
1、初末同辰，凡事宜速
2、魁罡所指，鬼神不能为祸
## 三传特性
- 初吉，末凶，先易后难；初凶，末吉，先难后易。凶，指的是刑冲破害、乘凶将等情况
- 占进取，三传五气需要逐渐增强；占脱散，三传五气要逐渐减弱。
- 初传和末传，初在日干寄宫前一位，末在日干寄宫后一位，叫前引后从，如果引从日干，主升官迁秩；引从地支，有迁居修宅
- 三传都阳，是显著，不怕人知道；三传都阴，事隐秘，不能被别人知道
- 三传为三合，或者初末六合，需要到日干与合住的地支相冲的时候，合才会冲开
- 三传进，宜进，凶吉也逐渐强烈；三传退，做事宜退，凶吉也逐渐消散
- 三传间隔，事多阻断
- 辰戌出现在三传，事情动荡不安，因为辰戌主暴躁
- 初凶，末克初，则逢凶化吉；初凶，还克末，就更凶了
- 末传克中传，叫“传中藏鬼”，事情会有转机，因为末为事情结果
- 三传都空，所占的事没有实际
- 当事的类神在三传中时，只以不为空的传来做出判断
- 末传为空，叫【传入空亡】，即使初中悲喜交加，解决也是一场空
- 三传互克，四课中凶神就无人制止，凶
- 初为死气煞，末为生气煞，吉利；初为生气煞，末为死气煞，不吉利
## 三传和四课的关系
- 三传都来于四课，叫【课内藏传】，所占的事，有成就之象。此时如果占病、诉讼等，就不好
- 日辰出现在三传，且三传为三合局，主事情反复不定，藕断丝连
- 三传都为父母爻，或者三合为父母爻，叫益气，所占的事助力多
- 三传都为妻财爻，或者三合为妻财爻，叫全财，财多身弱，不好
- 三传都为官鬼爻，或者三合为官鬼爻，叫全鬼，不吉利。如果四课上有制鬼之神，能转凶为吉
- 三传都为子孙爻，或者三合为子孙爻，叫全脱，消耗多，托多虚谎，谋事不成，求人费事儿
- 三传，初生中，中生末，末生日，叫顺生；末生中，中生初，初生日，叫逆生。主相求多人，低三下四，全凭他人之意
- 三传，初克中，中克末，末克日；或者末克中，中克初，初克日。主多方阻碍，从上到下都欺凌，事情都会失败，所求都没有
- 初为日阳神，末为辰阳神，说明是我托人办事；初为辰阳神，末为日阳神，说明是别人托我办事
- 三传为三合局时，若初为长生末为墓，主先易后难，初吉后凶；初为墓末为长生，主先难后易，初凶后吉
- 三传中，初末六冲，需要需要日干合住其中一个的时候，才能破解冲
- 三传，可以冲开四课中的墓库，但是只能一对一的冲
- 三传不在四课中，事难成
- 三传为日干寄宫，命主身强气盛；三传为日支，彼方强势
- 日干寄宫出现在三传，叫【三传朝日】，所占的事情，不管凶吉，都无法躲避
- 【夹定三传】，日上阳神、日上阴神，出现在初、中；干上阳神、干上阴神，出现在中、末；和窗口一样，三传在四课内前后滑动。所占事情，同样无法避免，且强度超过【三传朝日】
- 【夹定三传】时，如果窗口过大，会导致干上和支上夹了4个，超出了三传的三个，这种情况叫做【虚一】，就是不完美之意。虚的那一个支神，就是缺点。此时就看虚一的支神六亲是什么，乘了什么天官，来分析缺点是什么
- 【夹定三传】时，如果只夹住了两传，就叫【透关】。分析透出支神和日干的六亲，克被透出去了就是凶神已去，生被透出去了就是吉神没了。
- 初为日干墓，中为初墓，此时末能冲初中，就能破墓
- 中传为日墓，事情做到一半会终止；中传是日破、日冲，事情中途会有破坏；中传是日害，折腰，中阻；中为日空，断桥，事中断
- 三传，应该和日干进行比较，以得出六亲的类象
