# 八煞

分别是德、合、鬼、墓、刑、冲、破、害。其中德为庆会，合为完成，鬼主伤残，墓多暧昧，刑分强弱，冲不安宁，破知损坏，害见侵凌。旺相观察四时五气，空亡主一旬内的事，往来定远近之期，生克分父子之亲。
1、德
代表福报、机遇、贵人、和解、转危为安、祝福。总共有天德、月德、日德、支德。
日德，日德对自身影响最多：甲己日德在寅，乙庚日德在申，丙辛日德在巳，丁壬日德在亥，戊癸日德在巳。
支德，对家宅、内部、彼方影响最重：子日在巳，丑日在午，顺行十二辰。
2、合
主合作、和合、团结、羁绊。吉事逢合易成，凶事难解。因为除了和好，还有困住的意思。
干合：天干五合
支合：地支六合
三合：地支三合。
