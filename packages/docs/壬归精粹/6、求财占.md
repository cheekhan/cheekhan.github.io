# 求财占

财，为养身亲上之源。

求财占，主要讲类将的俯仰观察

## 类将

***选定类将***

断课第一步，就是选择正确的类将。类将，就是十二天官。

对于要占卜的事情，每个事情对应该对应一个或者多个类将。
例如玄武主盗贼丢失的事，青龙主管财，朱雀主合同证件考试等。

***类将的气***

选择好类将后，观察类将是否得气。
天神和天将互相克，就是神将不和，天将有忧无喜，这时天将都不得气。
其中，如果天神克天将，忧患比较严重；天将克天神，忧虑会轻。

天将无气，所求之事有忧虑。

## 类神

类神，就是天盘的支神，是类将所助的天神。类将能够提携、辅助类神。

***类神的气***

类神的气，由其所临地盘决定，类神所在地支是类神的十二长生状态。

如果类神临四吉气，则类神气旺；
临四凶，则类神气衰，临旬空也当凶处理；但是类神本身为旬空，下临不为空，此时出旬有望。

综合来说，神将即和，且类神下临吉地，则象吉；
神将不和，类神又下临凶地，则象凶。
如果类神临吉地，但是内战其将，也以凶象论。

类神下临十二长生，虽然大致的可以分为四吉、四平、四凶，但是对应到具体的十二长生，还是有细微的不同：

- 临【长生】：生生不息之义，如求财，利于投资取利等
- 临【沐浴】：败地，有清除减少的意思，无利可图
- 临【冠带】：光明正大，不可求阴私
- 临【临官】：禄地，荣华富贵，和官贵有关
- 临【帝旺】：旺地，大势所趋，乘势而起。但是要注意机不可失，且势不可使尽
- 临【衰】：逐渐消耗，需要尽早图取，晚了就没了
- 临【病】：此时气极弱
- 临【死】：生机断绝，没有希望
- 临【墓】：库地，隐藏、幽暗、暧昧不清。等待刑冲，才能破开阻碍
- 临【绝】：旧事已彻底完结，新事将来，开始看见希望了。需要等待蛰伏
- 临【胎】：气开始孕育发展，事情有希望，但是希望很弱小
- 临【养】：气开始壮大，需要积累，不可急躁

综合以上，由神将之间的状态，来决定所求的事是否顺利；天神自身的状态，决定了对天将的影响程度和性质。

## 类神阴神

***类阴对事情的凶吉影响***

类神所临地，决定了类神自身的状态是否好，也说明了事情的表象，例如所求的财，是一大笔财。

类神的阴神，说明了事情内在的问题，是事情的状况。和命主自身紧密相关，决定了从违缓急。

类阴，神司凶吉，将主其事。

如果阴乘吉将，气旺，生扶类将，则说明天心对类将有福佑的作用；
如果阴乘凶将，气囚，克制类将，则天意劫财，有耗损、夺走、分走的危险。

阴神生日干，同时生合日干上神，事情状况没有灾难；阴神克日干，并且刑害日上神，则不可妄动，财不属于你。

根据类阴，确定了凶吉，再看类阴乘之天官，就知道是什么事情带来的凶吉影响。如白虎，是疾病死丧，勾陈是争讼。
